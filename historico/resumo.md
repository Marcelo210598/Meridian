# Meridian.AI — Resumo Geral

## O que é
Dashboard multi-airdrop completamente separado do bot Ethereal.
Criado do zero como projeto independente em 2026-02-19.

## Arquitetura
- Meridian é autônomo: tem seu próprio banco, seu próprio Railway, seu próprio repo
- Bots e scripts enviam dados via `POST /api/ingest` com API Key por projeto
- Sem dependência de código com qualquer bot
